Show total sales for each woocommerce products.

== Description ==
Easily show total sales for each products on product pages and shop loop pages.
Enable/disable if you want to show only on product pages or just shop/category loop pages.
You can adjust the colors and font styles for both the pages.

If any issues regarding this plugin mail me @: sandeep.tete2013@gmail.com

== Installation ==
- Upload zip file.
- Activate the plugin.
- Scroll down to 'Total Sales For Woocommerce' and modify the settings.
- Done.
